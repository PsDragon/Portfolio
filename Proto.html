Proto Application

The prototype will be a demonstration of an application in which the computer vision algoritm is used as a fundamental feature. The problem my algorithm solves is given a set of car images and prior knowledge the user should be able to assign the label car to the images.

But what if you have a folder containing the images of cars, statues, dogs, mammals, and keyboards, and they are just placed random in the folder. Now imagine someone gives you a possible valid image of a mammal, and you want to look it up in the dataset if it´s correct. How would you do it?

The algorithm will be able to differentiate between a mammal, a dog, a car, a keyboard, and a statue. The algorithm will assign one of the labels (car, dog, keyboard, mammal, and statue) to the image through training previous images with a label. The algorithm analyses the visible characteristics of the images and learns about the classes.

When an image without a label comes as input into the model, the model analyses the visible characteristics (sex, dog fur colour, car colour, eye colour, etc.) of the image, and put a predicted label on it. The algorithm shows the confidence it has in being correct.

If the user wants to know how the algorithm interprets the image, the algorithm scanned the image through Grad-CAMs and shows the image with visual explanations about how the algorithm knows which label needs to be assigned to the image.

My algorithm contains a CNN network and has shown excellent results regarding image classification. Therefore, my algorithm can identify the images without labels.

Screens

Want to know more about the screens in the video? Well you're in luck! If you hover over the images, you will get explained what the purpose of that specific screen is.

Main Screen

The user will arrive at the main screen where buttons will be present which the user can interact with it to navigate through the application.

Algorithm Screen

If the user clicks on the algorithm button, the user will arrive at the algorithm screen where the user gets explained what the purpose of my algorithm is.

Upload Image Screen

If the user clicks on the upload image button, the user will arrive at the upload image screen where the user will drag an image to the image icon to get an animation of the upload of the image.

The Disruptive Technology Risk Screen

If the user clicks on the disruptive technology risk button, the user will arrive at the disruptive technology risk screen where the user gets explained what the privacy issues are concerning my algorithm.

XAI Screen

If the user clicks on the XAI button, the user will arrive at the XAI screen where the user will have to press the Grad-CAM button to change the image to Grad-CAM. There are multiple images that can be changed to Grad-CAM.

Predictions Screen

If the user clicks on the predictions button, the user will arrive at the predictions screen where the user will have to click on a button to show the predictions of an image. Here are also multiple predictions for multiple images.

Feedback Screen

If the user clicks on the feedback button, the user will arrive at the feedback screen where the user will be able to give feedback to the app. The user will be able to add comments and if the user fails to add comments, they can send an email via the email button.

The Algorithm

The prototype is based on my algorithm. The algorithm contains deep neural networks that can accurately classify an image while being sensitive to biases that may be present in the data the network is trained on. The algorithm had undergo a few steps:

  1. Data Exploration: Explore the Google Open Images dataset and discover biases if any are present
  2. Baseline Algorithm Design: Design and train a suitable baseline deep learning architecture
  3. Baseline Algorithm Improvement: Tune the hyperparameters of the model and use data pre-processing and data augmentation techniques if necessary to improve performance
  4. SOTA comparison: Compare the baseline models with SOTA (Transfer Learning & Fine Tuning)
  5. XAI: Explain the results of the AI algorithm and evaluate it's fairness, transparency, and interpretability
